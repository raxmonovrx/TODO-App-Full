# TODO App (Full Stack)

## Overview

A comprehensive TODO application with full-stack implementation.

## Features

- Task management
- User authentication
- CRUD operations
- Responsive design

## Installation

1. Clone the repository

```bash
git clone [https://github.com/raxmonovrx/TODO-App-Full]
cd TODO-App-Full
```

2. Install dependencies

````bash
# Frontend
cd frontend
npm install

## Usage
```bash

# Run
npm run start
````
